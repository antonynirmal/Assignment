Instructions and assumptions
1. Developed using spring boot, can be deployed in application server or run as stand alone app.
2. 3 projects are three - RestApp, WebServiceApp & EARbuilder
2. MySQL server is used as database.
3. JSON conversions are done internally in the Java code.
4. Max connections can be set in appliction sever level
5. WSDL url http://<host>:port/ws/gcd.wsdl
6. Sample request XML's are available in the WebserviceApp itself
